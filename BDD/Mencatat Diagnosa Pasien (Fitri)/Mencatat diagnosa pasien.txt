Feature : Mencatat rekam medis pasien yang sedang diperiksa
Skenario : Mencatat diagnosa pasien

Given Saya masuk pada antarmuka Rekam Medis Pasien
When Nama Dokter, Id Dokter, Id Rekam Medis, dan Tanggal hari ini sudah ditampilkan
Then Saya menginputkan Id Pasien, Keluhan Pasien, Pemeriksaan, Terapi, Alergi Obat,
     Kesimpulan Pemeriksaan, Kondisi Pasien Keluar
Then Saya mengklik tombol "Tambah Resep" untuk menambahkan resep obat untuk pasien
Then Saya memasukkan resep obat untuk pasien
When Saya mengklik tombol "save" saya mendapat pertanyaan "apakah data yang anda masukkan sudah benar?"
Then Saya memilih jawaban "Yes"
When Saya memilih jawaban "Yes"
Then Data Resep Obat tersimpan
And saya mendapatkan pesan informasi "data resep obat sudah tersimpan"
Then Saya memasukkan penyakit yang diderita pasien dari jComboBox
When Saya mengklik tombol "Save" saya mendapatkan pertanyaan "apakah data yang anda isi sudah benar?"
Then Saya memilih jawaban "Yes"
When Saya memilih jawaban "Yes"
Then Data diagnosa pasien tersimpan pada sistem
And Saya mendapatkan pesan informasi "data rekam medis sudah tersimpan"
