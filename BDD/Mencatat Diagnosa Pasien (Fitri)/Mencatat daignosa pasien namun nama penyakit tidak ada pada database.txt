Skenario : Mencatat diagnosa pasien namun nama penyakit tidak ada pada database

Given Saya masuk pada antarmuka Rekam Medis Pasien
When Saya memasukkan ID Pasien
Then Data identitas pasien ditampilkan
When Data Identitas pasien sudah ditampilkan
Then Field input keluhan pasien, diagnosa pasien, terapi, kesimpulan pemeriksaan, resep obat, dan nama penyakit "enable"
And Saya mengisi keluhan pasien, diagnosa pasien, terapi, kesimpulan pemeriksaan, dan obat
When Nama penyakit tidak ada pada database
Then Sistem mengeluarkan pesan "penyakit tidak ditemukan"
And Saya mencatat nama penyakit pada field daignosa pasien
When Saya mengklik tombol "save"
Then Data diagnosa pasien tersimpan pada sistem
