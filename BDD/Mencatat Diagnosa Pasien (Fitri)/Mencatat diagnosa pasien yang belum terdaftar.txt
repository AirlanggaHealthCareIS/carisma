Skenario : Mencatat diagnosa pasien yang belum terdaftar

Given Saya masuk pada antarmuka Rekam Medis Pasien
When Nama Dokter, Id Dokter, Id Rekam Medis, dan Tanggal hari ini sudah ditampilkan
Then Saya memasukkan ID Pasien
when ID pasien yang saya masukkan ysalah (belum pernah terdaftar)
Then Saya melihat pesan peringatan "Id Pasien xxxxx belum pernah terdaftar. Silakan mendaftar di bagian Resepsionis"
When Data identitas pasien tidak ditemukan
Then Saya tidak bisa mencatat diagnosa pasien
